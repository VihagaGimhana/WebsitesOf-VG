Headings

Headings are signposts that focus the reader on the most important content in a piece of writing, and are usually connected to the set question. "Provided they are well structured, a few headings make longer pieces" of writing easier to write and easier to read.

HTML contains 6 heading levels, "h1, h2, h3, h4, h5, h6".
Heading elements allow you to specify that certain parts of your content are headings or subheadings. In the same way that a book has the main title, chapter titles, and subtitles, an HTML document can too. But most of web designers commonly only use 3 to 4 at
Example: <>

<h1>This is my first editable heading</h1>

<h2>This is my first editable heading</h2>

<h3>This is my first editable heading</h3>

<h4>This is my first editable heading</h4>

<h5>This is my first editable heading</h5>
<h6>This is my first editable heading</h6>

Lists

Lists are useful because they: Highlight levels of importance. Help the reader understand the order in which things happen. Help readers skim and scan. There are 3 types of lists you can include in html.

Ordered List

These are sometimes called numbered lists because, by default, the list items contained in that list have a specific numerical order or ranking. Ordered lists are appropriate where the exact ordering of items are important to the meaning of the content. To create ordered list you need to start with "ol" element & for the each time you enter data you need to write in inside if "li" tag. here is an example.

  1. This is the first
  2. This is the second
  3. This is the third

Unodered List

An unordered list typically is a bulleted list of items. HTML 3.0 gives you the ability to customise the bullets, to do without bullets and to wrap list items horizontally or vertically for multicolumn lists.To create unordered list you need to start with "ul" element & for the each time you enter data you need to write in inside if "li" tag as you did in ordered list. here is an example.

  • This is the first
  • This is the second
  • This is the third

Tables

Tables are used to organize data that is too detailed or complicated to be described adequately in the text, allowing the reader to quickly see the results. They can be used to highlight trends or patterns in the data and to make a manuscript more readable by removing numeric data from the text. Tables have coloums and rows.
